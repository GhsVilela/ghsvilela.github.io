Hi, I'm Gustavo Vilela.

My career began as a developer, but after a while, I was given the opportunity to start as QA in a new company, to learn and work in this new area, since then, I have been working as a QA and have become more focused on automation because I have always enjoyed coding, being a test automator is my passion. I enjoy learning new tools and working with new programming languages. I believe that everyone should know the fundamentals of programming so that they can learn any new language. I'm a quick learner, if the tool that I need to work with has a "Get Started" documentation, I'll be able to learn it quickly, if not, I'll find a way, I never give up.
